## ğŸ“– Usage

### Add a Task

```bash

# Output: Created task #1: Buy groceries

# Output: Created task #2: Finish homework
```

### List All Tasks

```bash

# Output:
# ID  Status     Description
# 1   [ ]        Buy groceries
# 2   [ ]        Finish homework
# 2 tasks (0 completed, 2 pending)
```

### Complete a Task

```bash

# Output: Completed task #1: Buy groceries

# 1   [âœ“]        Buy groceries
```

```bash

# Output:

```

